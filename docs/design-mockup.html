Touch targets & accessibility baseline

  • Touch target minimum — 44 × 44pt (HIG). Exception: 9×9 cell on iPhone SE = 36pt (acknowledged HIG deviation; tap zone extended to cell border).
  • Dynamic Type — all non-cell text uses semantic SwiftUI fonts; spacing wrapped in @ScaledMetric; cell digit scales with cellSide, not type size; AX3 row stacks (S17).
  • Color-blind error — triple encoding: cell bg tint (#FBE3E1) + corner triangle shape (#C8362B) + digit color (#A52A20). Never red-only.
  • Reduce motion — pulse becomes static; cell place animation removed; shimmer becomes static fill.
  • Contrast (key pairs) — text.primary/surface.primary 16.1:1 AAA · text.user/cell.base 4.84:1 AA · accent.primary/surface.primary 4.84:1 AA · text.errorDigit/cell.error 4.7:1 AA.
