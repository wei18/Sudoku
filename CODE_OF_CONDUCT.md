# Code of Conduct

## Our pledge

We want participation in this project — issues, pull requests, discussions — to
be a harassment-free experience for everyone, regardless of background or
identity. We pledge to act in ways that keep the community open and welcoming.

## Our standards

Examples of behavior that contributes to a positive environment:

- Being respectful of differing viewpoints and experiences
- Giving and gracefully accepting constructive feedback
- Focusing on what is best for the project and its users

Unacceptable behavior includes harassment, insulting or derogatory comments,
personal or political attacks, publishing others' private information, and any
conduct that would reasonably be considered inappropriate in a professional
setting.

## Scope

This Code applies within all project spaces (the repository, issues, pull
requests, and any other project communication channel) and when an individual
is representing the project in public spaces.

## Enforcement

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1, as its standard for expected behavior.

To report a concern, contact the maintainer privately via the repository's
**Security** tab ("Report a vulnerability" opens a private channel) or by
opening an issue if the matter is not sensitive. The maintainer is responsible
for clarifying standards and will take appropriate, fair corrective action in
response to any behavior deemed inappropriate.
